Welcome to the **Sportchemy** documentation!

To include the Sportchemy extension in your Wowchemy website, simply import it in your `config/_default/config.yaml`:
```yaml
module:
  imports:
    # ... your other modules ...
    - path: github.com/Agos95/sportchemy/sportchemy
```
