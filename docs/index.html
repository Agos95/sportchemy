Welcome to the Sportchemy documentation!

To include the Sportchemy extension in your Wowchemy website, simply import it in your config/_default/config.yaml:

module:
  imports:
    # ... your other modules ...
    - path: github.com/Agos95/sportchemy/sportchemy

Table of Contents

  • New Page Archetypes

    New page archetypes which describe important events such as matches, and calendars.

  • Collection Block

    Sportchemy defines a new Collection block designed specifically for the match archetype. It also provides two different views to display the games.

  • Shortcodes

    Sportchemy introduces some additional shortcodes in addition to the Wowchemy ones.
